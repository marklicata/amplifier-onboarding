Comprehensive Code Review

Get a senior developer, QA engineer, and security expert to review your code in 3 minutes.

What It Does

This recipe performs multi-perspective code analysis by orchestrating three specialized AI agents: a senior architect, a bug hunter, and a security guardian. Each agent analyzes your code from their unique perspective, then the results are consolidated into a prioritized, actionable report.

Unlike single-perspective reviews, this recipe catches architectural issues, subtle bugs, and security vulnerabilities in one comprehensive pass - saving hours of manual review time.

How It Works

  1. Architectural Analysis (zen-architect)

    Evaluates code structure, design patterns, and architectural decisions. Identifies technical debt and suggests improvements.

  2. Bug Detection (bug-hunter)

    Systematically searches for logic errors, edge cases, and potential runtime issues. Uses hypothesis-driven debugging.

  3. Security Audit (security-guardian)

    Scans for vulnerabilities, insecure patterns, and compliance issues. Provides CVSS scores and remediation guidance.

  4. Consolidate Findings

    Combines all analyses, removes duplicates, and sorts by severity. Generates a unified report with clear priorities.

When to Use

  • Pre-commit quality checks - Catch issues before they reach the repository
  • Pull request reviews - Get thorough review faster than manual process
  • Legacy code assessment - Understand inherited code quality and risks
  • Security vulnerability scanning - Find security issues before production
  • Best practices validation - Ensure code follows language idioms and patterns

Required Inputs

file_path Required

Path to the file you want to analyze.

Example: src/auth.py

language Optional

Programming language (auto-detected if not specified).

Options: python, javascript, typescript, java, go

Default: python

focus_areas Optional

Specific areas to emphasize in the review.

Options: security, performance, style, architecture

Default: ["security", "architecture"]
