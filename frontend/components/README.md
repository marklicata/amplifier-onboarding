# Component Library

Reusable HTML components for the Amplifier web experience.

## Usage

Since we're using vanilla HTML (no framework yet), components are **templates** that you copy and customize.

Each component is documented with:
- HTML structure
- CSS classes (from `/styles/main.css`)
- Variants and options
- Usage examples
